# OPDeckBrain
![License: All Rights Reserved](https://img.shields.io/badge/license-All%20Rights%20Reserved-red)

OPDeckBrain is an advanced deck-building application dedicated to the
One Piece Card Game. The project provides analytical tools, AI-driven
deck suggestions, assisted deck generation, and detailed statistics to
help players optimize their lists.
